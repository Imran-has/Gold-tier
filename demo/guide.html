Before You Begin

Make sure you have the following ready before starting the setup:

  • A Windows PC or Laptop (Windows 10 or later)
  • Internet connection
  • Python installed (download from python.org — check "Add to PATH")
  • Facebook Business Page (with Admin access)
  • LinkedIn Account
  • Twitter/X Developer Account
  • Claude AI API Key (from console.anthropic.com)
💡 Note: If you do not have API keys yet, do not worry — Step 3 of this guide explains exactly where to get each one.
1
Download the System
STEP 1

Download from GitHub

Go to the link below and download the system files to your computer.

https://github.com/Imran-has/Gold-tier

On the GitHub page:

  • Click the green "Code" button
  • Click "Download ZIP"
  • Save the ZIP file to your Desktop
  • Right-click the ZIP → "Extract All"
  • Open the extracted folder "Gold-tier"
2
Run the Setup Wizard
STEP 2

Double-click setup.bat

Inside the Gold-tier folder, you will find a file called setup.bat.

  • Double-click setup.bat to run it
  • A black window (Command Prompt) will open
  • It will automatically install all required packages
  • Then it will ask you to enter your account details one by one
⚠️ Important: If Windows shows a security warning saying "Windows protected your PC", click "More info" then "Run anyway". This is normal for new applications.
3
Get Your API Keys

During setup, the wizard will ask for the following API keys and tokens. Here is where to get each one:

Platform What You Need Where to Get It
🤖 Claude AI API Key console.anthropic.com → API Keys
📘 Facebook Access Token + Page ID developers.facebook.com → My Apps → Tools
📸 Instagram Access Token + Account ID Facebook Developer Dashboard → Instagram Basic Display
🐦 Twitter/X API Key + Secret + Tokens developer.twitter.com → Projects & Apps
💼 LinkedIn Access Token linkedin.com/developers → My Apps
💡 Need help getting API keys? Contact support and we will guide you through the process via a 30-minute call at no extra charge.
4
Start Your AI Employee
STEP 4

Launch the System

Once setup is complete, open Command Prompt inside your Gold-tier folder and run:

python main.py

You will see your AI Employee start up and display a system readiness report showing which platforms are connected and ready.

TEST

Test Your Connections

To verify everything is working correctly, run:

python test_connections.py

You should see green checkmarks next to each platform confirming successful connections.

5
Daily Usage

Once the system is running, here is what happens automatically:

  • Every morning — AI posts content to all your social media platforms
  • Every week — Full financial report is generated and saved
  • Every Sunday at 9AM — CEO briefing report is created
  • Whenever a task fails — AI retries automatically up to 3 times
  • All activity is logged — Full audit trail available in the logs folder
💡 Pro Tip: Keep the system running in the background on your PC. For best results, ensure your computer stays on and connected to the internet.
